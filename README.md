This dashboard was designed for **job seekers, career transitioners, and job changers** to address a common issue: the data job market information is disorganized and difficult to understand. This project uses a *real-world dataset of 2024 data science job postings* (including titles, salary, and locations) to offer a singular, simple interface for analyzing market trends and compensation.

## Skills Showcased

- **Data Transformation (ETL) With Power Query:**
Cleaned, Shaped, and prepared the raw data for analysis by handling blanks, changing data types,and creating new columns.
- **Implicit Measures:**
Formulated measures to derive key insights and KPI's like 'Median Yearly Salary' and 'Job Count'.
- **Core charts:**
Utilized **Colomn, Bar, Line,** and **Area charts** to compare job counts and track trends over time.
- **Geospatial analysis:**
Leveraged **Map charts** to visualize the global distribution of jobs.
- **KPI Indicators & Tables:** 
Used **Cards** to display key metrics and **Tables** to provide detailed, sorted data.
- **Dashboard Design:** Created an engaging and visually striking layout, exploring both common and uncommon chart types to best tell the data story.
- **Interactive Reporting:** 
  - **Slicers:** To dynamically filter the report by job title.
  - **Buttons & Bookmarks:** To create an easy guiding experience.
  - **Drill Through:** To navigate from a high-level summary to a contextual, detailed view.
